# egzamino-uzduotis<br>
Išankstinė objektinio egzamino užduotis<br>
Yra duotos 4 užduotys, pagal kurias ir kursiu savo kodą. Trumpai aprašysiu kiekvieną iš jų remiantis savo kodo funkcionalumu.<br>

<b>Pirma užduotis</b><br>
1. Sukurtas bazinis kodo funkcionalumas.
2. Sukurtas failo nuskaitymas ir rezultatų išvedimas.
3. Suskaičiuojama kiek kiekvienas žodis pasikartojo nuskaitomame tekste ir žodis su pasikartojimų skaičiumi išsaugomi asociatyviame konteineryje <b>map</b>.
